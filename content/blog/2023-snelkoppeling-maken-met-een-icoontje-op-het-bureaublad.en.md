Part 1: creating the shortcut

1. Go to the website you find useful and want quick access to
   </br>For example: nu.nl , open in your browser: https:/www.nu.nl
2. Click (with the left mouse button) on the address bar where the website address is shown, it will turn blue.
   </br>If it does not automatically turn blue, you need to select the address and highlight it yourself
3. Right-click on the blue text and choose: Copy
4. Click the minus sign to minimise the web browser so you can see the desktop
5. Right-click on the desktop and choose: New -> Shortcut
6. Right-click in the location rectangle and choose: Paste
7. Click Next
8. Give the shortcut the name of the website and click: Finish
   </br></br>You now have your shortcut

Part 2: changing the icon to the website's logo

1. We are going to search Google for images of the logo
   </br>Search for example for: nu.nl logo
2. Click in the search engine on: Images
3. Right-click on the logo you like best and want to use and choose:</br>Open image in new tab
4. Click on the newly opened tab in the top menu of the web browser
5. Check that this is really the image of the logo you want to use as an icon on your desktop
6. Right-click on the logo and choose: </br>Save image as
7. Choose for example the Downloads folder
8. Note the file type shown under Save as and click: Save
   </br>For example as here: JPEG
9. Search on Google for a website to convert the image to ICO format
   </br>Search for example for: convert jpeg to ico
10. Choose a website that will convert the image format for you
    </br>Choose for example: https://www.freeconvert.com/ico-converter
    </br>Another website is perfectly fine too, as there are many of these comparable websites. </br>Just try one.
11. On the website click Choose Files and select your logo image from the folder where you saved it. </br>For example in: Downloads
12. Check that your Output is: ICO
13. Click the button on the website: Convert
14. Click the button on the website: Download
15. Click the minus sign to minimise the web browser so you can see the desktop
16. Right-click on your new shortcut and choose: Properties
17. Click the button: Change Icon... and then click: Browse
18. Go to the location where you saved the converted logo.
    </br>For example in: Downloads
19. Choose the logo that belongs to the website of the type: Icon and then click: Open
20. Click: OK, then: Apply, and then once more: OK
    </br></br>You now have the shortcut with a unique logo that belongs to the website
